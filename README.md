# Xenora

<p>
<b>Xenora</b> is a modern <b>Lost & Found platform</b> built with <b>React</b>.  
It connects people who <b>lost items</b> with people who <b>found them locally</b>, making recovery faster and easier.
</p>

<h2>🚀 Project Goal</h2>

<p>
Xenora aims to build a <b>trusted local lost-and-found ecosystem</b> where people can quickly report lost items, match them with found items nearby, and communicate safely to recover belongings.
</p>
